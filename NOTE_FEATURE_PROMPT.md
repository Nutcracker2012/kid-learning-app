## Requirements

### Content Requirements:
1. **Use simple, colloquial Chinese language**
   - Avoid complex or formal expressions
   - Use everyday vocabulary that children naturally use

2. **Single line format**
   - Must fit on one line and be suitable for printing on learning cards
   - Keep it concise and readable

3. **Specific, vivid, and observable features**
   - Features should be concrete and easy to identify visually
   - Use descriptive adjectives that paint a clear picture
   - Avoid abstract concepts

4. **Age-appropriate for 2-6 year olds**
   - Language level suitable for toddlers and preschoolers
   - Features should be things children can easily understand and remember
   - Use familiar concepts from their everyday experience

5. **Relaxed, fun, and friendly tone**
   - Make it engaging and playful
   - Should feel like a game or puzzle
   - Encourage curiosity and memorization

### Display Requirements:
- Label: "Note:" (in bold/SemiBold)
- Content: Single line formula showing 3 key features + equals sign + Chinese name
- Position: After pronunciation section, before recognition features (if shown)
- Typography: Regular body text size (16px), left-aligned

## Implementation Notes:
- This replaces or complements the detailed recognition features list
- The formula should highlight the 3 most distinctive and memorable features
- Features are connected with "+" signs and end with "=" followed by the Chinese name
- The entire note should be a memory aid that makes identification fun and easy
